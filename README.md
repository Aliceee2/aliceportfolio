# UI/UX Portfolio

Готовый сайт-портфолио с режимом редактирования прямо в браузере.

## Как запустить

1. Откройте файл `index.html` в браузере.
2. Нажмите кнопку `Режим редактирования`.
3. Редактируйте тексты прямо на странице.
4. Добавляйте/удаляйте проекты через кнопки в карточках.

## Хранение данных

- Данные сохраняются автоматически в `localStorage`.
- Кнопка `Экспорт JSON` делает бэкап контента.
- Кнопка `Импорт JSON` восстанавливает контент из файла.

## Что поменять сразу под себя

- Имя, описание и контакты в хедере и блоке контактов.
- Карточки проектов (название, описание, ссылка, изображение).
- Услуги можно менять в `script.js` в массиве `services`.
